# Project SFG (Stefano-Federico-Giulia)

Progetto SAndroidE che consente di leggere i dati variabili di un circuito con resistenza variabile ed inviarli tramite Bluetooth ad una applicazione Android.
Il circuito è formato da un RedBear BLE Nano e da un trimmer.
L'applicazione android riceve i dati dal BLE Nano e li visualizza sullo schermo dello smartphone.
